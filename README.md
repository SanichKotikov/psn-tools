# psnTools

Desktop application for tracking prices in PSN.

Using: [Chart.js](http://www.chartjs.org/) (simple HTML5 Charts using the canvas element)

![app](./screenshots/app.png)

### Build

By default, this build osx64 app only.

```bash
$ npm run build
```
